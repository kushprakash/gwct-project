2. USER CREATION FLOW

---

Village User Creation Logic:

If Login User = State User:
Select:
Channel → District → Block → Panchayat
Then Create Village User

If Login User = Channel Partner:
Select:
District → Block → Panchayat
Then Create Village User

If Login User = District User:
Select:
Block → Panchayat
Then Create Village User

If Login User = Block User:
Select:
Panchayat
Then Create Village User

If Login User = Panchayat User:
Direct Create Village User

Same hierarchy logic for:

* Panchayat User Creation
* Block User Creation
* District User Creation
* Channel Partner Creation
